## Features

<CardGrid stagger>
	<Card title="Cross Platform">
		Runs on windows,linux,Mac OS and various game devices such as PSP,miyoo,RG35xx.
	</Card>
	<Card title="Open Source">
		Access [the source code](https://github.com/djdiskmachine/LittleGPTracker) under the CC-BY-NC-SA license. 
	</Card>
	<Card title="Refined interface">
		It implements the user interface of littlesounddj, a very famous tracker for the Gameboy platform that has been tried and tested by many users over the years.
	</Card>
	<Card title="Free">
		You can get the binaries [here](https://github.com/djdiskmachine/LittleGPTracker/releases).
	</Card>
</CardGrid>
